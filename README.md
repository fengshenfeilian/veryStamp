# VeryStamp

### Description

VeryStamp(好打印)是一个深度整合线下打印业务的线上平台。

打印场景：

|              | VeryStamp | 线下打印店 |
| ------------ | --------- | ---------- |
| 纸质打印资料 | 不需要    | 可能需要   |
| 移动存储设备 | 不需要    | 需要       |
| 排队等候     | 不需要    | 可能需要   |
| 24小时打印   | 可以      | 不可以     |
| 配送服务     | 可以      | 不可以     |
